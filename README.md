- get some docker json from docker inspect, this is actually a 1-element arry, so we get an annoying _1 variable

```
g@unit01:~/code/lash/lib/jstruct$ JSON_STRING=$(docker inspect $(docker ps -q | head -n 1) ) decodeJson
```
